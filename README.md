# opsplash
A command line tool unpack/repack oppo/realme/oneplus splash image

## How to compile
### Main program
*** Need zlib-devel    
``` sh
make
```
### As python library
``` sh
python3 setup.py build
python3 setup.py install
```

## Prebuilt library
In prebuilt dir    
Have win64 python3.10 library    
Have win64 dll library
